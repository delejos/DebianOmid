About / درباره

چرا دبی‌امید؟

Why DebiOmid?

«امید» یعنی باور به فردایی روشن‌تر. این پروژه برای هر ایرانی ساخته شده — در هر کجای دنیا — تا بتواند با زبان مادری‌اش از لینوکس استفاده کند.

دبی‌امید یک اسکریپت نصب خودکار است که دبیان ۱۳ (Trixie) را به‌طور کامل برای کاربران فارسی‌زبان فارسی می‌کند: فونت، کیبورد، تقویم جلالی و بسته‌های زبانی — همه از مخازن رسمی دبیان.

امید
omid  /ɒˈmiːd/  — Persian
hope  ·  aspiration

"Omid" means hope. This project is a small contribution — making sure every Iranian, wherever they are, can use Linux in their own language.

DebiOmid automates the full Farsi localization of Debian 13 (Trixie): fonts, keyboard input, Jalali calendar, and language packs — installed entirely from official Debian repositories.

The Lion and Sun (شیر و خورشید) is one of the oldest Persian symbols — rooted in ancient astronomy, representing the sun in the house of Leo. It has been part of Persian identity for thousands of years, long before any political association.

Features / ویژگی‌ها

چه چیزی نصب می‌شود؟

What gets installed and configured?

🔤
فونت‌های فارسی
Persian Fonts
نصب fonts-vazirmatn و fonts-noto-core برای نمایش بی‌نقص متون فارسی در همه برنامه‌ها.
⌨️
کیبورد فارسی خودکار
Auto-configured Keyboard
Fcitx5 با موتور m17n (استاندارد ISIRI 2901) نصب و پیکربندی می‌شود. کلید Alt+Shift به‌صورت خودکار تنظیم می‌شود — بدون نیاز به تنظیم دستی.
📅
تقویم جلالی
Jalali Calendar
پشتیبانی از تقویم شمسی در GNOME، KDE Plasma 6 و XFCE.
🌐
برنامه‌های فارسی
Localized Apps
بسته‌های زبان فارسی برای Firefox-ESR و LibreOffice — رابط کاربری هر دو برنامه به فارسی.
🛡️
بررسی نسخه دبیان
Version Check
اسکریپت نسخه سیستم را بررسی می‌کند و اگر دبیان ۱۳ نباشد، قبل از ادامه تأیید می‌گیرد.
📋
گزارش نصب کامل
Full Install Log
تمام خروجی در /var/log/debiomid-install.log ذخیره می‌شود تا عیب‌یابی آسان باشد.

Install / نصب

شروع کنید

One command to get started

یک دستور. فقط همین.

$ wget -O- https://raw.githubusercontent.com/delejos/DebiOmid/main/setup.sh | sudo bash
  • دستور بالا را در ترمینال اجرا کنید.
  • پس از اتمام، از سیستم خارج شده و دوباره وارد شوید.
  • کیبورد فارسی (Alt+Shift) از پیش تنظیم شده است.
  • کاربران گنوم: Extension Manager را باز کنید و «Persian Calendar» را نصب کنید.
  • در صورت مشکل، فایل /var/log/debiomid-install.log را بررسی کنید.
Uninstall / حذف
$ wget -O- https://raw.githubusercontent.com/delejos/DebiOmid/main/uninstall.sh | sudo bash
